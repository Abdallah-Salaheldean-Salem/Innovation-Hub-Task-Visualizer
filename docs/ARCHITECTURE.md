# Architecture

This document describes the architecture of the application.

## Frontend
- **React 18+** with TypeScript
- **Vite** for fast build tooling
- **Tailwind CSS** for styling
- **Lucide React** for icons

## State Management
- Local React State and Context
- Local Storage for persistence

## Component Structure
- `App.tsx`: Main entry point and layout shell
- `components/`: Feature-based components (KanbanBoard, TimelineView, IdeasView, etc.)
- `types.ts`: TypeScript definitions
